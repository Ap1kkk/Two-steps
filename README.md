## 🚀 О проекте

**Sport Line** — это полноценное клиент-серверное веб-приложение, объединяющее спорт и туризм. Проект позволяет пользователям исследовать местность, прокладывать спортивные маршруты, делиться достижениями, оставлять отзывы о локациях и находить единомышленников.

### Ключевые особенности:
- 🔐 **Аутентификация и авторизация** с использованием JWT-токенов
- 🗺️ **Интерактивные карты** для визуализации маршрутов
- 📊 **Персональный кабинет** с историей активностей
- 👥 **Социальные функции**: добавление в друзья, обмен маршрутами
- 📱 **Адаптивный дизайн** для всех типов устройств

---

## ✨ Основные возможности

### Frontend
- 🎨 **Современный UI** на React 18 с хуками и функциональными компонентами
- 🗺️ **Визуализация маршрутов** с помощью Яндекс.Карт
- 📱 **Адаптивная верстка** с использованием SCSS-модулей
- 🔄 **Маршрутизация** с React Router DOM v7
- 🧹 **Автоматическое форматирование** кода через ESLint, Stylelint и Prettier

### Backend
- 🔐 **JWT-аутентификация** с refresh-токенами
- 👤 **Управление пользователями**: регистрация, профили, роли (USER/ADMIN)
- 🗺️ **CRUD операции** над маршрутами и локациями
- ⭐ **Система оценок и отзывов** к маршрутам
- 📈 **Статистика активности** пользователей
- 🗄️ **PostgreSQL** для хранения данных, **H2** для разработки

## 🛠 Технологический стек

### Frontend

| Категория | Технологии |
| :--- | :--- |
| **Ядро** | React 18, React Router DOM 7, TypeScript 5 |
| **Сборка** | Webpack 5, TS Loader, PostCSS, Sass (SCSS) |
| **Линтеры** | ESLint, Stylelint, Prettier |
| **Карты** | @pbe/react-yandex-maps (Яндекс.Карты) |
| **HTTP Client** | Fetch API / Axios |
| **Утилиты** | clsx, cross-env |

### Backend

| Категория | Технологии |
| :--- | :--- |
| **Фреймворк** | Spring Boot 3.x |
| **Безопасность** | Spring Security, JWT (jjwt) |
| **База данных** | PostgreSQL (продакшн), H2 (разработка) |
| **ORM** | Spring Data JPA, Hibernate |
| **Сборка** | Maven / Gradle |
| **Документация API** | Swagger / OpenAPI 3 |
| **Тестирование** | JUnit 5, Mockito, Spring Boot Test |
| **Логирование** | SLF4J + Logback |

## 💻 Начало работы

### Требования

**Общие:**
- Git
- IDE (IntelliJ IDEA, VS Code и др.)

**Frontend:**
- Node.js: `24.14.0` (LTS) или выше
- npm: `10.x` или выше

**Backend:**
- JDK 17 или 21
- Maven 3.8+ / Gradle 8.x
- PostgreSQL 15+ (для продакшн-окружения)
